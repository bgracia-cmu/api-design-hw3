## Description
An API for playing a Connect 4 game.

## Running the text client example:
1. Run
```sh
./start.sh
```

or

```sh
javac *.java
java TextClient
```

## Viewing the API docs:

1. Run
```sh
./create-javadoc.sh
```

2. Open the generated `docs/index.html` website in a web browser

3. Navigate to `ConnectFourJSON` interface
